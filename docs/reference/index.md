# Reference

What each part of Ymir does. Complete and dry.

- **[Nodes](nodes/index.md)**. Every node, grouped by category, with its inputs, outputs, parameters, and layer contract. Generated from the running build, so it always matches the current nodes.
- **[World settings](world-settings.md)**. The project-wide values: seed, world extent and height, sea level, resolution.
- **[Export formats](export-formats.md)**. The three heightmap outputs: 16-bit PNG, raw R16, and 32-bit EXR.
- **[Project format](project-format.md)**. The `.ymir` save file.
- **[Command line](cli.md)**. The headless `ymir-cli` runner.
