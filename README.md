# Ymir

An open-source, node-based procedural terrain generator for Linux.
Everything is a layered field; nodes just transform it.

Named for the primordial giant of Norse myth, whose body the world is shaped from.

## Status

Early development. Not yet usable.

## Building

Requires a Rust toolchain via [rustup](https://rustup.rs). The pinned compiler
version lives in `rust-toolchain.toml` and is fetched automatically.

```bash
cargo build
```
